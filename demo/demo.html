SmartView - Demo

SmartView is able to read html fragments specfied by a selector and create models out of them. This is a naive process where the view, on initialize, will read all tags with a data attribute. The attribute will be of the following format:

data-attr=key value

This example will add { key: "value" } to the SmartView's model. When the SmartView is finished divining the model, a "ready" event will be posted with the model as an argument. The view will additionally be bound to the Model's change event.
